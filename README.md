# TMF Sandbox Website

This repository contains the code for the simple static sandbox website used in the TMF Scholar IT Fundamentals assignment.

## Purpose
- Host a basic static website (Home, Intake Form, Contact pages).
- Practice deployment to Netlify/Vercel.
- Serve as a live example for systems thinking and troubleshooting exercises.

## Boundaries (Sandbox Only)
- Purely for training purposes.
- Use only fake/placeholder data in the intake form.
- No real user data or production features.

## Sandbox Rules
- All changes must be documented.
- Broken scenarios will be intentionally introduced for troubleshooting practice.
- This repo is connected to a free-tier hosting service (Netlify/Vercel).

## Deployment
The site is deployed via Netlify/Vercel and linked to this GitHub repository for automatic deployments.
